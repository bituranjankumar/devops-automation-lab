# Monitoring with AWS CloudWatch  

This directory contains examples and configuration files for getting started with AWS CloudWatch for monitoring and logging.  

Included examples:  

- `alarms.tf`  : Terraform code to create a simple CPU utilization alarm for EC2.  
- `dashboard.json`  : A sample CloudWatch dashboard definition to visualize metrics.  
- `README.md`  : This file describing the examples and usage.  

These examples are targeted at beginners and focus on basic metrics, logs, and alarms. Customize them for your AWS environment.
