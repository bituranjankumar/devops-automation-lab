# AWS EC2 Beginner Guide

This directory contains basics for getting started with Amazon EC2.

Included examples:

- `ec2-instance.tf`: Terraform example for provisioning a `t2.micro` EC2 instance in the ap-southeast-1 (Singapore) region.  
- `user-data.sh`: Sample user data script that installs updates and a web server.  
- `README.md`: This file explaining the example and how to apply/destroy it.  

These examples are intended for the AWS free tier and provide simple starting points you can customize.
