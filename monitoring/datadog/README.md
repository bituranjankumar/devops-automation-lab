# Monitoring with DataDog  

This directory contains examples and configuration files for getting started with DataDog for monitoring and logging.  

Included examples:  

- `dashboard.json`  : A sample DataDog dashboard definition to visualize metrics.  
- `monitor.yml`  : Example monitor configuration to alert on CPU usage.  
- `README.md`  : This file describing the examples and usage.  

These examples are introductory and focus on setting up basic metrics, logs, and monitors. Customize them to suit your infrastructure.
